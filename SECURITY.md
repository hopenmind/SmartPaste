# Security Policy

## Supported Versions

Currently, the `main` branch of Smart Paste is supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| v1.0.x  | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within Smart Paste, please DO NOT report it on the public issue tracker.

Instead, please send an email to the security team or maintainers at Hope 'n Mind (https://www.hopenmind.com).

Please include the following information in your report:
- Type of vulnerability (e.g., privilege escalation, remote code execution).
- Steps to reproduce the vulnerability.
- Expected vs. actual behavior.
- Any additional context or screenshots.

We will endeavor to respond to your report within 48 hours. If the vulnerability is confirmed, we will work to patch it and release an update as quickly as possible.
