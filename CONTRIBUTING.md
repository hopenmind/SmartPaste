# Contributing to Smart Paste

First off, thank you for considering contributing to Smart Paste! It's people like you that make open-source software such a great community.

## Where do I go from here?

If you've noticed a bug or have a feature request, make sure to check our [Issues](#) to see if someone else has already created a ticket. If not, go ahead and [make one](#)!

## Fork & Pull Request Workflow

1. **Fork** the repo on GitHub.
2. **Clone** the project to your own machine.
3. **Commit** changes to your own branch.
4. **Push** your work back up to your fork.
5. Submit a **Pull request** so that we can review your changes.

NOTE: Be sure to merge the latest from "upstream" before making a pull request!

## Development Setup

- Ensure you have **.NET 8 SDK** installed.
- Open `SmartPaste.sln` (or the `.csproj`) in Visual Studio or Visual Studio Code.
- Build the project to restore NuGet packages.

## Code of Conduct

Please note that this project is released with a Contributor Code of Conduct. By participating in this project you agree to abide by its terms. Let's build a welcoming and inclusive community!
